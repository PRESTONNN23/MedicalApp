# MedicalApp

Простое веб-приложение для учета пациентов и их посещений.

## Функционал
Добавление пациентов
Просмотр списка пациентов
Добавление посещений
Экспорт данных пациента в XML

## Технологии
ASP.NET Core MVC
Entity Framework Core
SQLite
jQuery + AJAX

## Запуск

```bash
dotnet restore
dotnet run

Открыть в браузере:
http://localhost:5077
